# Event Registration

## Tools

Go to Main Menu -> Tools

* TCN Event Reports
* TCN Event Inspector

## A User With Just A Phone Number

* Main Menu -> Tools
* TCN Add Phone Number Only Registrant
* Select an event
* Enter a phone number
* Enter a name
* Click "Add User"

When you go to the event inspect page, you'll see the user:

http://cvnet.jonhill.ca/
