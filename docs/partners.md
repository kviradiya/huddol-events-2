# Network Partners

## Viewing

* Log in to the admin panel
* Select "Users" from the main menu
* Click on "Network Partners" above the list of Users
* URL should now be http://cvnet.jonhill.ca/wp-admin/users.php?role=network_partner (bookmark if you like!)
* Select a Network Partner
* You can edit
    * name
    * image
    * biographical information
    
## Partner in Profile

* Select the current partner in profile
* uncheck "Partner in Profile" near the bottom of the profile screen
* Select the network partner you wish to edit, as above in "Viewing"
* add "Partner in Profile" near the bottom of their profile screen
